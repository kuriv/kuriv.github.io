执行运算符

PHP 支持一个执行运算符 ` 。注意这不是单引号！ PHP 将尝试将反引号中的内容作为 Shell 命令来执行，并将其输出信息返回（即可以赋给一个变量而不是简单地丢弃到标准输出）。使用执行运算符 ` 的效果与 shell_exec() 函数相同。

<?php

$foo = `ping 61.139.2.69 -n 3`;
var_dump($foo);
