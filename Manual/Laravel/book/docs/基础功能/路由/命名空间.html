命名空间

可以使用下面的方法分配同一个 PHP 命名空间给该分组下的多个控制器。

<?php

Route::namespace('Admin')->group(function () {
    Route::get('/', 'IndexController@index');
    Route::get('user', 'UserController@index');
});
