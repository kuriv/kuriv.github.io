# 路由前缀

你可以为分组中所有路由 URI 添加给定的前缀。

```php
<?php

Route::prefix('user')->group(function () {
    Route::get('/', function () {
        return route('user');
    })->name('user');
    Route::get('{id}', function ($id) {
        return route('profile', [
            'id' => $id
        ]);
    })->name('profile');
});

```
