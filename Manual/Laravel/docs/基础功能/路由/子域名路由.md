# 子域名路由

路由分组还可以被用于处理子域名路由，子域名可以像 URI 一样被分配给路由参数，从而允许捕获子域名的部分用于路由或者控制器。

```php
<?php

Route::domain('{domain}.laravel.com')->group(function () {
    Route::get('/', function ($domain) {
        return $domain;
    });
    Route::get('user/{id}', function ($domain, $id) {
        return $domain . '-' . $id;
    });
});

```
