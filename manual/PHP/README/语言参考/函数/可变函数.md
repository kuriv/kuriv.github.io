### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

$foo = 'foo';

/**
 * Just a test function.
 *
 * @param  void
 * @return void
 */
function foo()
{
}

$foo();

```

附例二：

```php
<?php

declare(strict_types = 1);

$foo = 'foo';

/**
 * Just a test function.
 *
 * @param  string $string
 * @return void
 */
function foo(string $string)
{
}

$foo('foo');

```

附例三：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return void
     */
    public function method()
    {
    }
}

$foo = new Foo;
$func = 'method';
$foo->$func();

```

附例四：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test property.
     *
     * @var string
     */
    public static $property = 'foo';

    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return 'bar';
    }
}

var_dump(Foo::$property);   // string(3) "foo"

$property = 'method';
var_dump(Foo::$property()); // string(3) "bar"

```
