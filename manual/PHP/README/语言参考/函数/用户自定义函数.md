### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

/**
 * Just a test function.
 *
 * @param  void
 * @return void
 */
function foo()
{
}

```

附例二：

```php
<?php

declare(strict_types = 1);

foo();

/**
 * Just a test function.
 *
 * @param  void
 * @return void
 */
function foo()
{
}

$condition = true;

if ($condition) {
    /**
     * Just another test function.
     *
     * @param  void
     * @return void
     */
    function bar()
    {
    }
}

bar();

```

附例三：

```php
<?php

declare(strict_types = 1);

/**
 * Just a test function.
 *
 * @param  void
 * @return void
 */
function foo()
{
    /**
     * Just another test function.
     *
     * @param  void
     * @return void
     */
    function bar()
    {
    }
}

foo();
bar();

```

附例四：

```php
<?php

declare(strict_types = 1);

/**
 * Recursive function.
 *
 * @param  int    $number
 * @return void
 */
function foo(int $number)
{
    if ($number < 10) {
        foo($number + 1);
    }
}

foo(1);

```
