### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

define('FOO', 'foo');
define('BAR', 'bar');

var_dump(FOO); // string(3) "foo"
var_dump(BAR); // string(3) "bar"

```

附例二：

```php
<?php

declare(strict_types = 1);

const FOO = 'foo';
const BAR = 'bar';

var_dump(FOO); // string(3) "foo"
var_dump(BAR); // string(3) "bar"

```
