## 魔术常量

### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

var_dump(__LINE__); // int(5)

```

附例二：

```php
<?php

declare(strict_types = 1);

var_dump(__FILE__);

```

附例三：

```php
<?php

declare(strict_types = 1);

var_dump(__DIR__);

```

附例四：

```php
<?php

declare(strict_types = 1);

/**
 * Just a test function.
 *
 * @param  void
 * @return string
 */
function foo(): string
{
    return __FUNCTION__;
}

var_dump(foo()); // string(3) "foo"

```

附例五：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return __CLASS__;
    }
}

var_dump(Foo::method()); // string(3) "Foo"

```

附例六：

```php
<?php

declare(strict_types = 1);

trait FooTrait
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return __TRAIT__;
    }
}

class Foo
{
    use FooTrait;
}

var_dump(Foo::method()); // string(8) "FooTrait"

```

附例七：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return __METHOD__;
    }
}

var_dump(Foo::method()); // string(11) "Foo::method"

```

附例八：

```php
<?php

declare(strict_types = 1);

namespace Foo;

class Foo
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return __NAMESPACE__;
    }
}

var_dump(Foo::method()); // string(3) "Foo"

```
