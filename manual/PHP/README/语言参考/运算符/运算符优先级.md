## 运算符优先级

| 结合方向 |                          运算符                          |
| :------: | :------------------------------------------------------: |
|    无    |                        clone new                         |
|    左    |                            [                             |
|    右    |                            **                            |
|    右    | ++ -- ~ (int) (float) (string) (array) (object) (bool) @ |
|    无    |                        instanceof                        |
|    右    |                            !                             |
|    左    |                          * / %                           |
|    左    |                          + - .                           |
|    左    |                          << >>                           |
|    无    |                        < <= > >=                         |
|    无    |                   == != <> === !== <=>                   |
|    左    |                            &                             |
|    左    |                            ^                             |
|    左    |                          &#124;                          |
|    左    |                            &&                            |
|    左    |                       &#124;&#124;                       |
|    左    |                            ??                            |
|    左    |                            ?:                            |
|    右    |     = += -= \*= /= %= \*\*= .= &= &#124;= ^= <<= >>=     |
|    左    |                           and                            |
|    左    |                           xor                            |
|    左    |                            or                            |

附例一：

```php
<?php

declare(strict_types = 1);

$foo = 3 * 3 % 5;

var_dump($foo); // int(4)

```

附例二：

```php
<?php

declare(strict_types = 1);

$foo = true ? 0 : true ? 1 : 2;

var_dump($foo); // int(2)

```

附例三：

```php
<?php

declare(strict_types = 1);

$foo = 1;
$bar = 2;
$foo = $bar += 3;

var_dump($foo); // int(5)
var_dump($bar); // int(5)

```

附例四：

```php
<?php

declare(strict_types = 1);

$foo = 1;
$bar = $foo + $foo++;

var_dump($bar); // int(3)

```
