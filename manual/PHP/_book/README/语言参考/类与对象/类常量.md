## 类常量

可以把在当前类中始终保持不变的值定义为常量。在定义和使用常量的时候不需要使用 `$` 符号。

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test constant.
     *
     * @var string
     */
    const CONSTANT = 'foo';

    /**
     * Just a test function.
     *
     * @param  void
     * @return string
     */
    public function method(): string
    {
        return self::CONSTANT;
    }
}

$foo = new Foo;

var_dump(Foo::CONSTANT);  // string(3) "foo"
var_dump($foo::CONSTANT); // string(3) "foo"
var_dump($foo->method()); // string(3) "foo"

```

类常量与常量不同的是，类常量可以被子类重写：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test constant.
     *
     * @var string
     */
    const CONSTANT = 'foo';
}

class Bar extends Foo
{
    /**
     * Override the parent constant.
     *
     * @var string
     */
    const CONSTANT = 'bar';
}

class Baz extends Foo
{
    /**
     * Override the parent constant.
     *
     * @var string
     */
    const CONSTANT = 'baz';
}

var_dump(Foo::CONSTANT); // string(3) "foo"
var_dump(Bar::CONSTANT); // string(3) "bar"
var_dump(Baz::CONSTANT); // string(3) "baz"

```
