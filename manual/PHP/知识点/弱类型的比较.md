## 类型转换

附例一：

```php
<?php

declare(strict_types = 1);

var_dump(0 == '');

```

输出：

```
bool(true)
```

附例二：

```php
<?php

declare(strict_types = 1);

var_dump(null == false);

```

输出：

```
bool(true)
```

附例三：

```php
<?php

declare(strict_types = 1);

var_dump(1 == '1abc');

```

输出：

```
bool(true)
```

附例四：

```php
<?php

declare(strict_types = 1);

var_dump(0 == 'abc');

```

输出：

```
bool(true)
```
