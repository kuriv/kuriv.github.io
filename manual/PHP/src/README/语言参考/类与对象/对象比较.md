## 对象比较

* [基本语法](#基本语法)

### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test property.
     *
     * @var string
     */
    public $property;

    /**
     * Just a test function.
     *
     * @param  void
     * @return void
     */
    public function method()
    {
    }
}

$foo = new Foo;
$bar = new Foo;

var_dump($foo == $bar);  // bool(true)
var_dump($foo != $bar);  // bool(false)
var_dump($foo === $bar); // bool(false)
var_dump($foo !== $bar); // bool(true)

```

附例二：

```php
<?php

declare(strict_types = 1);

class Foo

    /**
```
