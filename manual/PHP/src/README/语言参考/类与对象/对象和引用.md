### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test property.
     *
     * @var string
     */
    public $property = 'foo';
}

$foo = new Foo;
$bar = $foo;
$bar->property = 'bar';

var_dump($foo->property); // string(3) "bar"

```

附例二：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test property.
     *
     * @var string
     */
    public $property = 'foo';
}

$foo = new Foo;
$bar = &$foo;
$bar->property = 'bar';

var_dump($foo->property); // string(3) "bar"

```

附例三：

```php
<?php

declare(strict_types = 1);

/**
 * Just a test function.
 *
 * @param  Foo    $instance
 * @return void
 */
function foo(Foo $instance)
{
    $instance->property = 'bar';
}

class Foo
{
    /**
     * Just a test property.
     *
     * @var string
     */
    public $property = 'foo';
}

$foo = new Foo;
foo($foo);

var_dump($foo->property); // string(3) "bar"

```
