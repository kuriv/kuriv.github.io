附例十五：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test property.
     *
     * @var string
     */
    public $property = 'foo';

    /**
     * This method is called when tries to dump an object.
     *
     * @param  void
     * @return array
     */
    public function __debugInfo(): array
    {
        return [
            'property' => 'bar'
        ];
    }
}

$foo = new Foo;

var_dump($foo); // object(Foo)#1 (1) { ["property"]=> string(3) "bar" }

```
