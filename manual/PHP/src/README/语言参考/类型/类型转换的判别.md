## 类型转换的判别

* [基本语法](#基本语法)

### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

$foo = '2';
$foo *= 2;

var_dump($foo); // int(4)

```

附例二：

```php
<?php

declare(strict_types = 1);

$foo = 2;
$foo *= 1.3;

var_dump($foo); // float(2.6)

```

附例三：

```php
<?php

declare(strict_types = 1);

$foo = 5 * '10foo';

var_dump($foo); // int(50)

```
