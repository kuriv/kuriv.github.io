## 魔术常量

获取文件中的当前行号：

```php
<?php

declare(strict_types = 1);

var_dump(__LINE__); // int(5)

```

获取当前文件的完整路径和文件名：

```php
<?php

declare(strict_types = 1);

var_dump(__FILE__);

```

获取当前文件所在的目录：

```php
<?php

declare(strict_types = 1);

var_dump(__DIR__);

```

获取当前函数的名称：

```php
<?php

declare(strict_types = 1);

/**
 * Just a test function.
 *
 * @param  void
 * @return string
 */
function foo(): string
{
    return __FUNCTION__;
}

var_dump(foo()); // string(3) "foo"

```

获取当前类的名称：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return __CLASS__;
    }
}

var_dump(Foo::method()); // string(3) "Foo"

```

获取当前 `Trait` 的名称：

```php
<?php

declare(strict_types = 1);

trait FooTrait
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return __TRAIT__;
    }
}

class Foo
{
    use FooTrait;
}

var_dump(Foo::method()); // string(8) "FooTrait"

```

获取当前类的方法名：

```php
<?php

declare(strict_types = 1);

class Foo
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return __METHOD__;
    }
}

var_dump(Foo::method()); // string(11) "Foo::method"

```

获取当前命名空间的名称：

```php
<?php

declare(strict_types = 1);

namespace Foo;

class Bar
{
    /**
     * Just a test method.
     *
     * @param  void
     * @return string
     */
    public static function method(): string
    {
        return __NAMESPACE__;
    }
}

var_dump(Bar::method()); // string(3) "Foo"

```
