### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

$foo = 'foo';
$Foo = 'Foo';

var_dump($foo); // string(3) "foo"
var_dump($Foo); // string(3) "Foo"

```

附例二：

```php
<?php

declare(strict_types = 1);

$_foo = 'foo';

var_dump($_foo); // string(3) "foo"

```

附例三：

```php
<?php

declare(strict_types = 1);

$_中文 = 'foo';

var_dump($_中文); // string(3) "foo"

```
