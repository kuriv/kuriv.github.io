### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

require __DIR__ . '/example.php';

var_dump($foo); // string(3) "foo"

```

example.php

```php
<?php

declare(strict_types = 1);

$foo = 'foo';

```

附例二：

```php
<?php

declare(strict_types = 1);

$foo = 'foo';

/**
 * Just a test function.
 *
 * @param  void
 * @return void
 */
function foo()
{
    return $foo;
}

var_dump(foo()); // NULL

```

附例三：

```php
<?php

declare(strict_types = 1);

$foo = 'foo';

/**
 * Just a test function.
 *
 * @param  void
 * @return string
 */
function foo(): string
{
    global $foo;
    return $foo;
}

var_dump(foo()); // string(3) "foo"

```

附例四：

```php
<?php

declare(strict_types = 1);

$foo = 'foo';

/**
 * Just a test function.
 *
 * @param  void
 * @return string
 */
function foo(): string
{
    return $GLOBALS['foo'];
}

var_dump(foo()); // string(3) "foo"

```

附例五：

```php
<?php

declare(strict_types = 1);

/**
 * Just a test function.
 *
 * @param  void
 * @return int
 */
function foo(): int
{
    static $foo = 0;
    return $foo++;
}

var_dump(foo()); // int(0)
var_dump(foo()); // int(1)
var_dump(foo()); // int(2)

```

附例六：

```php
<?php

declare(strict_types = 1);

/**
 * Just a test function.
 *
 * @param  void
 * @return void
 */
function foo()
{
    static $foo = 0;
    $foo++;
    if ($foo < 10) {
        foo();
    }
}

foo();

```
