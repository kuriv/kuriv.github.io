### 基本语法

附例一：

```php
<?php

declare(strict_types = 1);

$foo = 9.5;

var_dump(-$foo); // float(-9.5)

```

附例二：

```php
<?php

declare(strict_types = 1);

$foo = 9.5;
$bar = 4;

var_dump($foo + $bar); // float(13.5)

```

附例三：

```php
<?php

declare(strict_types = 1);

$bar = 4;

```
