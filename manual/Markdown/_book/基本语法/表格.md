## 表格

语法：

```markdown
| 字段1 | 字段2 |
| ---- | ---- |
| 单元1 | 单元1 |
| 单元2 | 单元2 |
```

效果：

| 字段1 | 字段2 |
| ----- | ----- |
| 单元1 | 单元1 |
| 单元2 | 单元2 |

语法：

```markdown
|   左对齐   |   居中对齐   |   右对齐   |
| :--------- | :--------: | ---------: |
| $1         | $100000000 |      $1000 |
| $10        | $10000     |        $10 |
| $1000      | $1         |         $1 |
```

效果：

| 左对齐 |  居中对齐  | 右对齐 |
| :----- | :--------: | -----: |
| $1     | $100000000 |  $1000 |
| $10    |   $10000   |    $10 |
| $1000  |     $1     |     $1 |
