# 版本约束

版本约束可以用几个不同的方法来指定。

| 名称   | 实例                  | 描述                             |
| ------ | --------------------- | -------------------------------- |
| 版本号 | `1.0.2`               | 指定版本                         |
| 范围   | `>=1.0,<2.0`          | 指定范围版本                     |
| 通配符 | `1.0.*`               | `>=1.0,<1.1`                     |
| 波浪号 | `~1.2` <br/> `~1.2.3` | `>=1.2,<2.0` <br/>`>=1.2.3,<1.3` |
| 折音号 | `^1.2` <br/> `^1.2.3` | `>=1.2,<2.0` <br/>`>=1.2.3,<2.0` |
